# TASKER

A Web Application that keep the record of the tasks and allows user to <br>
- create <br>
- read <br>
- update <br>
- delete <br>

# BACKEND
To run backend use Commands <br>
 => cd backend <br>
 => npm install <br>
 => npm start <br>
 
 # FRONTEND
 To run Frontend use Commands <br>
  => cd frontend <br>
  => npm install <br>
  => npm start <br>
